MC-PTT (재난안전통신단말기)

MC-PTT
(재난안전통신단말기)

제품 개요

재난 관련 기관의 무선통신망을 통합한 국가무선통신망으로, 재난 발생 시 일사불란한 지휘통제가 이루어질 수 있으며

통신시스템의 신뢰성과 안전성, 보안성 및 즉각 대응성이 보장됩니다.

주요 기능

제품 특징

IP-PTT Server
  • • 1:1 , 1:N 음성 PTT
  • • 1:1 , 1:N 영상 PTT
  • • 일제지령에 의한 1:N PTT
  • • 녹취 및 녹화 연동
Call Bridge Server
  • • 음성 및 영상 VolP
  • • 가입자별 등급 지정
  • • IP PBX 부가기능
    (당겨받기/돌려주기/착신전화)
  • • FMC 연동 기능
